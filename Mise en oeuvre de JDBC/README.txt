Java Database Connectivity

Le but est de se connecter à une base de données locale à travers une application JAVA standalone.
Afin d’y parvenir, j’ai utilisé :
- MySQL (le driver correspondant – que vous trouverez d’ailleurs téléchargé – a été ajouté aux « Modules »)
- WampServer pour créer la BDD et la visualiser avant et après exécution du code
- IntelliJ IDEA pour écrire le code et le tester
